# 📌 Sistema de Gestión – Pura Dulzura

Aplicación de escritorio desarrollada en C# y .NET para la gestión integral de una empresa, incluyendo clientes, pedidos, stock y empleados.

---

## 🚀 Tecnologías utilizadas

* C#
* .NET
* XML
* POO
* Arquitectura en capas (Entity, DAL, BLL, Mapper, Seguridad, Servicios, UI)
* Git

---

## ⚙️ Funcionalidades principales

* ✔ Gestión y administración de clientes
* ✔ Gestión de pedidos y emisión de factura en PDF
* ✔ Administración de Perfiles, usuarios y contraseñas.
* ✔ Administración de empleados
* ✔ Carga de novedades y liquidación de sueldos con envio por e-mail.
* ✔ Validaciones y lógica de negocio
* ✔ Persistencia de datos con XML
  
---

## 🧱 Arquitectura

El sistema está diseñado utilizando **arquitectura en capas**, separando responsabilidades para mejorar la escalabilidad y el mantenimiento:

- **Entity** → Modelos de datos utilizados en el sistema  
- **DAL (Data Access Layer)** → Acceso a datos y persistencia  
- **BLL (Business Logic Layer)** → Lógica de negocio y reglas del sistema  
- **Mapper** → Conversión y mapeo entre estructuras de datos (por ejemplo, listas a DataTable y adaptación entre capas)  
- **Seguridad** → Validación de usuarios y gestión de autenticación (login, verificación de credenciales encriptadas)  
- **Services** → Funcionalidades reutilizables como encriptación/desencriptación, cálculo de CUIL y utilidades generales  
- **UI** → Interfaz de usuario y presentación (formularios y manejo de interacción con el usuario) 

---

## 💡 Características destacadas

* Diseño modular orientado a escalabilidad
* Separación clara de responsabilidades
* Implementación de buenas prácticas de desarrollo
